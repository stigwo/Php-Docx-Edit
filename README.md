# Php-Docx-Edit
Simply replace strings in a *.docx by PHP

Upload all files to a subfolder called /doc

Generated files are placed in /doc/gen
Original .docx file is in /doc/org folder

$replacements in  original non.docx is
'NAVNKINA' 
'NAVNNORGE'
'DATESIGN' 
'MANUFCONTACT' 
'BUYERCONTACT'
